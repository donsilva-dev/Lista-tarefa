# 📋 Lista de Tarefas Animada — Flutter

Aplicativo moderno de lista de tarefas desenvolvido em Flutter utilizando arquitetura organizada com **Model + Controller**, animações suaves e interface moderna.

---

# ✨ Funcionalidades

✅ Adicionar tarefas  
✅ Editar tarefas  
✅ Excluir tarefas  
✅ Marcar como concluída  
✅ Swipe para excluir  
✅ BottomSheet animado  
✅ Microinterações  
✅ Cards modernos  
✅ Arquitetura organizada    
✅ Responsividade

# 📦 Como executar o projeto

## Clone o repositório

```bash
git clone https://github.com/seuusuario/animated_task_app.git
```

---

## Entre na pasta

```bash
cd animated_task_app
```

---

## Instale as dependências

```bash
flutter pub get
```

---

## Execute o projeto

```bash
flutter run
```
